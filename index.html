üîê Privacy Policy

Your privacy is important to us. This policy outlines how WavyBot collects, uses, and protects your data.

1. Data Collection

WavyBot collects only the data strictly necessary for its functionality, including:

  • Discord User IDs: To identify users for moderation actions, command usage, and personalized features (e.g., leveling system). These are stored encrypted.
  • Discord Guild (Server) IDs: To store server-specific configurations (e.g., logging channels, anti-raid settings) and track bot presence.
  • Discord Channel IDs: For setting up specific channels for logs, welcome messages, etc.
  • Limited Message Content: Temporarily processed for command parsing, anti-raid detection, and moderation logs. Message content is NOT stored unless explicitly required for a specific bot feature (like suggestion logs, which would be visible to your server staff) and is always anonymized where possible.
  • Bot-related Settings: Configurations set by server administrators (e.g., enable/disable features, thresholds, custom messages).

2. Use of Data

The collected data is used exclusively for:

  • Providing WavyBot's intended features (moderation, anti-raid, utility, fun commands).
  • Storing and applying server-specific configurations.
  • Internal analytics to improve bot performance and identify common issues (anonymous).
  • Responding to support requests.

3. Data Storage & Security

All data is stored securely using industry-standard practices. We take reasonable measures to protect your information from unauthorized access, alteration, disclosure, or destruction. Data is stored on secure servers located [mention rough region if known, e.g., "in North America"]. Data is retained only as long as necessary to provide the bot's services or as required by law.

4. Data Sharing

WavyBot does NOT share, sell, rent, or trade your personal data with any third parties under any circumstances.

5. Your Rights

You have the right to request access to the data WavyBot holds about your server or your user ID. You can also request the deletion of your server's configuration data by removing WavyBot from your server or by contacting our support team via our Support Server.
